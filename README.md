# PDF 분석 기반 인터랙티브 Q&A 시스템

이 애플리케이션은 사용자가 PDF 보고서를 업로드하면 AI가 내용을 분석하고, 사용자가 선택한 텍스트에 대해 심층적인 질문과 답변을 제공하는 도구입니다.

## 설치 및 실행 방법

1.  **가상 환경 생성 (선택 사항)**
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # Mac/Linux
    source venv/bin/activate
    ```

2.  **의존성 설치**
    ```bash
    pip install -r requirements.txt
    ```

3.  **환경 변수 설정**
    - `.env.example` 파일의 이름을 `.env`로 변경합니다.
    - `.env` 파일을 열어 `OPENAI_API_KEY`에 본인의 OpenAI API 키를 입력합니다.
    ```
    OPENAI_API_KEY=sk-...
    ```

4.  **애플리케이션 실행**
    ```bash
    python app.py
    ```

5.  **사용하기**
    - 웹 브라우저에서 `http://127.0.0.1:5000`으로 접속합니다.
    - PDF 파일을 업로드합니다.
    - 자동 생성된 요약과 키워드를 확인합니다.
    - PDF 본문에서 궁금한 문장이나 단락을 드래그하여 선택합니다.
    - 우측 패널에 생성된 심화 질문을 클릭하여 답변을 확인합니다.

## 주요 기능
- **PDF 뷰어**: PDF.js를 사용하여 브라우저 내에서 PDF 렌더링.
- **AI 요약**: 문서 전체 내용을 요약하고 핵심 키워드 추출.
- **인터랙티브 Q&A**: 텍스트 드래그 시 관련 질문 자동 생성 및 답변 제공.

## 주의사항
- PDF 파일의 텍스트가 추출 가능한 형태여야 합니다 (이미지 스캔본은 인식되지 않을 수 있음).
- OpenAI API 사용량에 따라 요감이 발생할 수 있습니다.
